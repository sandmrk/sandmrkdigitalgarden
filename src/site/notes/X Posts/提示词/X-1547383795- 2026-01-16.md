# 通用提示词：

### 💡 核心观点
有意思！用AI生成属于自己的“人生仪表盘”！！！ 最近朋友公司有个项目做了一个智慧系统，说白了就是一个好看的数据仪表盘，领导来参观的时候，可以展示的，有没有实际用处不知道，但感觉花了小100W，搞出来的就是一个仅供展示的“花瓶”。 朋友把做出来的阶段性的试用链接发给我，让我提点意见，还真是好看，看起来特专业，做了数字孪生，首页界面还有个AI助手，可以对话让它介绍项目的方方面面。 还真别说，我要是投资方或者ZF领导，还挺喜欢这种看起来“高级”的界面，一览全局的感觉。 我感觉这种系统，技术原理不复杂啊，通过智能硬件监测数据，回传到数据库然后通过UI展示出来，或者再搞几个算法模型弄点智能预测、安全预警之类的“花哨功能”，这也值100W？？ 更何况，这些公司做了那么多，很多底层框架直接套用就行了，总感觉有50W都是智商税。 扯远了，这种仪表盘还真是不错，所以我想着可以搞出一个人生仪表盘，从健康、财务、家庭、个人成长，职业发展、技能学习等全方面的数据可视化，搞出一个炫酷的人生系统，也挺有意思的。 所以肝了一下午，认真“拆解”了一下自己，搞出来一个“沐阳人生仪表盘”，但因为涉及隐私数据太多，就不分享了，另外生成了两个“可分享版”，朋友们可以看看效果… 提示词还需要再完善一下，因为要生成属于自己的仪表盘，需要填入的信息太多，需要精简精简。

### 🧵 深度展开
> [!NOTE] 补充 1
> 通用提示词： 创建一个全面的人生仪表盘信息图，采用简洁科技化的瑞士设计风格，包含以下模块： ## 设计要求 - 风格：现代科技感 + 瑞士国际主义设计（简洁、网格化、高信息密度） - 配色：科技蓝/青色系主色调 + 荧光强调色 + 黑白灰，使用渐变色块区分模块 - 布局：严格网格系统，模块化分区，左上角或顶部中心放置头像区 - 字体：无衬线字体（如 DIN、Helvetica），清晰层级数字使用等宽字体 - 图表：简洁图表类型（条形图、环形图、时间轴、雷达图、仪表盘） - 科技元素：细线框、HUD界面感、数据流线条、半透明面板 ## 头像与身份区（重点模块） ### 头像设计 - 位置：左上角或顶部居中，作为视觉焦点 - 样式：圆形或六边形头像框 - 边框：发光边框效果或数据环绕效果 - 尺寸：占整体版面的8-12% - 科技感处理：  - 头像周围添加扫描线圈动效  - 或添加数据光环（显示关键数值）  - 或使用渐变边框 + 阴影效果 ### 身份信息面板（紧邻头像） - 姓名（大号字体） - 年龄 + 生日 - 职业/身份标签 - 个人座右铭/Slogan（可选） - 已生存：X天 X小时 X分钟（实时感数字） - 预期剩余天数（基于80岁寿命） - **人生进度条**（视觉化百分比，带渐变填充） ## 核心数据模块（8-12个） ### 1. 生命时钟仪表盘 - 中心圆形仪表盘显示人生完成度 - 环形进度条：已度过/剩余时间 - 周围卫星数据：  - 本年进度  - 本月进度  - 本周进度 ### 2. 时间资产矩阵 - 时间分配环形图：睡眠/工作/学习/娱乐/社交/通勤 - 每周168小时热力图 - 时间投资效率指数 - 专注时间累计计数器 ### 3. 健康监测面板 - HUD风格生命体征显示 - 体重/BMI 折线图（带趋势箭头） - 运动数据：年度步数总计、锻炼天数、卡路里消耗 - 睡眠质量雷达图 - 健康评分仪表盘（0-100分） ### 4. 财务控制台 - 资产总览仪表盘 - 收入vs支出对比柱状图 - 储蓄率进度环 - 财务自由度百分比（带目标线） - 被动收入占比 ### 5. 技能矩阵 - 六边形技能雷达图（6-8个维度） - 技能等级显示（Lv.1-10） - 正在升级的技能进度条 - 专业领域深度指标 ### 6. 社交网络图谱 - 关系网络节点图（核心圈/紧密圈/泛社交圈） - 人际互动频率热力条 - 关键关系维护质量评分 - 社交能量平衡指数 ### 7. 成就系统 - 垂直时间轴展示里程碑事件 - 成就徽章墙（图标化展示） - 年度标完成度（多进度条） - 解锁成就计数器 ### 8. 知识库统计 - 年度阅读：书籍数 + 总页数 - 知识领域分布饼图 - 学习时长累计仪表 - 笔记/输出数量 ### 9. 足迹地图 - 简化世界地图 + 打卡标记 - 去过城市/国家计数 - 旅行距离总里程 - 新体验尝试次数 ### 10. 情绪分析仪 - 30天情绪波动曲线 - 情绪分布占比（积极/中性/消极） - 压力指数仪表盘（0-10） - 幸福感评分（带星级显示） ### 11. 创造力输出 - 创作项目计数器 - 产出类型分布图（写作/设计/代码/视频等） - 创意实现率进度环 - GitHub风格贡献热力图 ### 12. 习惯追踪器 - 核心习惯打卡热力图（365天视图） - TOP 5习惯坚持天数排行 - 习惯完成率环形图 - 连续打卡最长记录 ## 视觉与科技化元素 ### 界面装饰 - 细线网格背景 - 数据连接线（连接头像与各模块） - 半透明卡片面板 - 发光效果（关键数据） - 扫描线动效暗示 ### 图标系统 - 统一的线性图标 - 每个模块配专属图标 - 图标 + 数据的组合展示 ### 数据可视化 - 渐变色填充 - 数据点高亮 - 对比色标注异常值 - 趋势箭头指示 ## 布局参考结构  ┌─────────────────────────────────────────────────┐ │ [头像] 姓名 年龄 生命进度条 │ │ 身份标签 已生存X天 │ ├──────────┬──────────┬──────────┬───────────────┤ │ 时间资产 │ 健康面板 │ 财务控制 │成就系统 │ ├──────────┼──────────┴──────────┤│ │ 技能矩阵 │ 情绪分析仪 │(时间轴) │ ├──────────┼──────────┬──────────┼───────────────┤ │ 社交网络 │ 知识库│ 足迹地图 │习惯追踪 │ └──────────┴──────────┴──────────┴───────────────┘  ## 输出规格 - 全中文标注 - 比例：16:9 或 A3横向 - 分辨率：至少 3840×2160 (4K) - 格式：PNG/SVG（支持矢量缩放） - 可导出为：打印海报 / 桌面壁纸 / 数字展示 ## 使用指令 请生成一个 [32]岁 [男性] 的人生仪表盘信息图： - 头像风格：[极简剪影] - 重点领域：[职业发展] - 配色偏好：[简约黑白/渐变彩色] - 特殊要求：[添加具体数据/年度对比/季度回顾]
