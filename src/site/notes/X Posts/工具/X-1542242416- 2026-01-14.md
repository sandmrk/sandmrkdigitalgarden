# X 上的 周览资源：“想整理微信聊天记录时，很多工具要么上传数据到云端，要么导出格式混乱，隐私风险也让人担心。 推荐这个开源项目： EchoTrace，完全在本地运行，可以安全导出、分析微信聊天记录，还能生成年度报告。数据不出电脑，既能回顾聊天内容，也能生成统计、趋势等分析图表，非常适合做个人归档或年度总结。 https://t.co/zXXMWOnb42” / X
